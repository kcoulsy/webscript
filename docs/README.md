# WebScript Docs

Docusaurus site for WebScript documentation.

## Local development

```bash
pnpm install
pnpm start
```

## Build

```bash
pnpm build
```

## Structure

| Folder | Contents |
|--------|----------|
| `docs/intro.md` | Overview |
| `docs/tutorial/` | Step-by-step app tutorial |
| `docs/language/` | Syntax, types, directives, server logic |
| `docs/stdlib/` | Standard library helpers |
| `docs/guides/` | In-depth framework guides |

Edit files under `docs/docs/`.
